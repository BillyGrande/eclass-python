Variables

There is no specific command to declare a variable in Python. You only need the name of the variable, the '=' sign and the value of the variable. The name of the variable cannot start with a number. 


x = 10
name = "Bill"
shop_list = ["bananas","milk","peanut-butter"]

Variable names are case-sensitive 

            
a = 1
A = "soft bread"

print(a) # 1
print(A) # soft bread
# a and A variables are not the same

Varibles can be a whole expression 

            
x = (2+3)/2
y = 1 == 2
z = range(10,100,10)
