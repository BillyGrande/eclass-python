Conditionals

Conditionals add logic and flow to our programs. The main way to python conditions is If statements.

If

Before we start with if statements, firs we should know about logical conditions and operators.

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b
With that in mind we can move on with the if statement.

With if statement, if the condition is True, we will execute the block of the if, else we will skip it. 


a = 1
b = 10
if b > a:
  print("b is greater than a") # The print will be executed because b>a is True

Else

With else if the value of the if statement is false, we will execute the else block. 


a = 100
b = 10
if b > a:
  print("b is greater than a") # This won't be executed
else:
  print("b is not greater than a") This will be

Elif

We can add more if statements after our first if statment, using the elif keyword. If the first if statement if False, then the first elif statement that is true will be executed. If none elif statement is true if we have an else statement, then the else block will be executed. 


a = 10
b = 10

if b > a:
  print("b is greater than a") # This won't be executed
elif b == a:
  print("b is equal to a") # This will be executed
elif b < a:
  print("b is not greater than a") # This will be skipped, because the first elif got executed
else:
  print("How did you end up here?")

As you can see, we can have multiple elifs.
