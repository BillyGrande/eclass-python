Numbers

Python has 3 data types for numbers. Integer, float and complex

Integer

Int, or integer, is whole a whole number, without decimals. It can be postive or negative. Int type is unbounded, that means theoritacally it can be of unlimited length. 

            
x = 1
y = 99999999999999
z = -324

Float

Float number can be a positive or negative number that contains one or more decimals. 


x = 3.14
y = 1.0
z = -23.12

Complex

Complex numbers are written with a "j" as the imaginary part: 


x = 3+5j
y = 2j
z = -5j
