Text

For having text in python we use string and char data types.

String

In Python, a string is a sequence of characters enclosed within single quotes, double quotes, or triple quotes. Strings are immutable, meaning once they are created, their contents cannot be changed

Strings can include letters, numbers, symbols, and even spaces. They are often used to represent text and are one of the most commonly used data types in Python. 

 
#Single quotes
single_quote_string = 'Hello, World!'

#Double quotes
double_quite_string = "Hello, World?"

#Triple quotes (can spn mutliple lines)
triple_quote_strings = '''
Hello,
World
'''

As you can see, triple quote strings can span multiple lines

Character

Character or char type is a string of length one. 


# Single character
char_a = "a"
char_b = 'b'
