Other types

Some types we missed, but are useful for our adveture are:

  • Boolean
  • None
  • Byte

Boolean

Boolean or bool is the True of False values. The comparison of two values will be a boolean result. Boolean type always start with a capitilized letter. 


x = True
x = 1 > 0 # Will be True

y = False
y = 1 > 2 # Will be False

None

None is when we have an empty/null value. None starts with a capital N. 


x = None

Byte

You can create Bytes usin the byte() function or b prefix. Bytes are useful for specific occasions that we will not cover here. 


data_bytes = b"Hello, Bytes!"
