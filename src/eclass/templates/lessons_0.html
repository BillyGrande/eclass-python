What is python

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.

Python history

Python first released in 1991 and it was designed by Guido van Rossum.

Dynamic typing

Dynamic type is a type system where the program doesn't need to have explicit type for variables. For example in a statically typed language if you want to initialize an integer variable you would type something as 


int my_variable = 0
but in a dynamically typed language, like python, you can omit the type and write it as this 

my_variable = 0
A lot of programmers prefer statically typed languages because they are easier to debug but dynamically typed languages are faster to write

Basic Syntax

The purpose of this section is just a quick heads-up about basic python syntax. We will dive deeper for each topic in later chapters.

Variables

To initialize a variable just use an alphanumeric sequence and = operator. Variables cant start with number. 


my_variable = 0
v4ri4ble = "Test"

You can change the value of a variable with = operator again. 


variable_0 = 0
print(variable_0)
#Output: 0

variable_0 = 1
print(variable_0)
#Output: 1

Print

You can print in output using the ~print()~ function. 


print("This is a message")
#Output: This is a message

number = 5
print(number)
#Output: 5

print("The value is ", number)
#Output: The value is 5
