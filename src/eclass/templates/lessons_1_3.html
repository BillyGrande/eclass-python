Set and Dicts

Sets and sets have many similarities with sequence but their main difference is they have unique items.

Set

Sets are like lists but they must have unique items. They can't have duplicates, but we are able to modify their items. Also their items cannot be other immutable types like list.

To initialize a set we use '{}' 


my_set = {1,2,"hello"} # No duplicate values

Dict

Dicts are like sets but they consit of key:value pairs for times. Keys cannot be duplicates. Also dicts are changeable. To initialize a dict we use '{}' too, but its items must follow the notion of key:value 


my_dict = {
  "Tutorial": "Pyton",
  "one" : 1,
   1 : "one",
   "fruits" : [apple, banana, orange]
}

print(my_dict["one"]) # Will print 1
print(my_dict[1]) # Will print "one"
print(my_dict[fruits]) # Will print the list [apple, banana, orange]
