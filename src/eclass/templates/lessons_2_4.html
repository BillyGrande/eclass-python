Libraries

Libriaries is a collection of already defined functions, already programmed that we can use in our code.

Libraries are useful because we can used complex functions without the need to program them.

To use a library we use the import keyword and the library name, we import libraries preferably at the start of the program. 


# Let's calculate the square root of 16
import math # We import math library

A = 16
print(math.sqrt(A))
# We use the library name and the function we want, sqrt() in our case
# This will print 4

We can only import a specif function of a library using from keyword. 

            
# Importing specific items
from math import sqrt, sin

A = 16
B = 3.14
print(sqrt(A))
print(sin(B))
