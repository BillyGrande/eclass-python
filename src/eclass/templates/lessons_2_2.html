Loops

Loops help us to write less code and repeat part of our code if a statement is True.

We have 2 kinds of loops:

  • While loops
  • For loops

While

With while loop, we can execute a set of statements as lon as condition is True. 


i = 1
while i < 6:
  print(i)
  i += 1

# This will print 1,2,3,4,5 in order.

Break

With break keyword we can stop the loop even if the while condition is True. 


i = 1
while i < 6:
  if i == 3:
    break
  print(i)
  i += 1

# This will print 1,2 and then leave the loop.

Continue

With continue keyword we skip the current iteration and continue with next one. 


i = 1
while i < 6:
  if i == 3:
    continue
  print(i)
  i += 1

# This will print 1,2,4,5. It will skip number 3.

For

For loops are very common in python. A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string). This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages. With the for loop we can execute a set of statements, once for each item in a list, tuple, set etc. 


fruits = ["apple", "banana", "cherry"]
for x in fruits:
  print(x)

# This will print apple, banana, cherry in order.
        

        
for x in "apple":
  print(x)

# This will print a,p,p,l,e in order.

Range

To loop through a set of code a specified number of times, we can use the range() function 


for x in range(6):
  print(x)

# Thsi will prin 1,2,3,4,5 in order.

Pass

For loops cannot be empty, but if you for some reason have a for loop with no content, put in the pass statement to avoid getting an error. 


for x in [0, 1, 2]:
  pass

Break and Continue

Like the while loop we can use the break and continue keywords in for loops too. 


fruits = ["apple", "banana", "cherry"]
for x in fruits:
  if x == "banana":
    break
  print(x)

# This will print apple.

 fruits = ["apple", "banana", "cherry"]
for x in fruits:
  if x == "banana":
    continue
  print(x)

# This will print apple, cherry.
